Generation Technology Characterstics Computer

  1. Machine Language
  2. Huge size
  3. High consumption of Electricity
  4. Highly expensive
  1. ENIAC
  2. UNIVAC
  3. EDVAC
  4. EDSAC
::Second Generation(1959-65) Transistor Based
  1. Batch Processing,Multiprogramming OS
  2. Expensive
  3. FORTRAN,COBOL
  1. IBM 1620
  2. IBM 7094
  3. CDC 1604
  4. CDC 3600

  1. IC used
  2. Smaller size
  3. Faster
  4. Generated less heat
  5. Costly
  1. IBM-360 series
  2. Honeywell-6000 series
  3. PDP(Personal Data Processor)
  4. TDC-316
